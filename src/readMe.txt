** Directions for running the Pipe and Filters Program **

Note: All java and text input files are contained in this zip file. 

1.) Once the package has been unzipped, execute the command below to compile the Java code:

    "javac *.java"

2.) This will create all of the Java classes.
3.) In order to run the program, execute the command below in a terminal session:

    "java App"

4.) Once that is running, the program will prompt the user for what text file they want to run.
5.) Enter the number 1-3 depending on which file you want to use
